> ### DATABASE OPERATIONS

> #### How do you create a database?

```sql
CREATE DATABASE store;
```

> #### How do you select a database?

```sql
USE store;
```

> #### How do you delete a database?

```sql
DROP DATABASE store;
```

> #### How do you rename a database?

```sql
ALTER DATABASE store MODIFY NAME = stores
```
